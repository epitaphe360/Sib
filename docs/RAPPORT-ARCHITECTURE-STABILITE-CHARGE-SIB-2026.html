1. Synthèse exécutive

La plateforme SIB 2026 repose sur une architecture cloud moderne en trois couches : Vercel (site web), Supabase (données, authentification, API, scans), Railway (services backend complémentaires). Cette combinaison a été dimensionnée pour un grand salon (plan Pro Supabase avec Compute Large, Vercel Pro, Railway Pro).

Résultats des tests de charge (production) :

  • 200 000 scans QR exécutés en ~9 minutes, 0 % d'erreur, 100 % stockés en base
  • 200 portiques simultanés validés sans échec
  • 5 000 lectures API simultanées sans erreur
  • Site public supporté par un CDN mondial (Vercel Edge)

Conclusion : l'architecture est adaptée à une affluence de 200 000 visiteurs, avec badges imprimés en volume et au moins 200 utilisateurs QR app en parallèle.

3. Supabase — cœur de la plateforme

3.1 Pourquoi Supabase ?

Supabase regroupe en un seul service managé ce qu'un salon professionnel nécessite : base PostgreSQL, API REST + RPC, authentification unifiée web + mobile, Row Level Security (RLS), Edge Functions, Realtime et Storage. Un seul backend pour web et mobile évite la duplication et simplifie la maintenance pendant l'événement.

3.2 Plan choisi : Pro + Compute Large (Scaling)

CaractéristiqueBénéfice salon
Plan ProPas de limites du plan gratuit (connexions, CPU, bande passante)
Compute LargePlus de CPU/RAM pour pics d'entrée et stats admin
ScalingMontée en charge automatique sous affluence
EgressBande passante pour images, API, app mobile

Coût BC infrastructure : 6 300,20 MAD HT / 3 mois (déboursé).

3.3 Tables et RPC critiques

3.4 Sécurité

  • RLS sur toutes les tables sensibles
  • JWT côté serveur pour badges (secret jamais dans l'app)
  • RPC SECURITY DEFINER pour scans
  • Policies dédiées staff sur access_logs

4.2 Pourquoi le site reste stable sous affluence

  1. Pages publiques = fichiers statiques après build
  2. CDN Vercel absorbe des milliers de requêtes/s sans toucher l'origine
  3. Séparation des charges : vitrine ≠ scans (Supabase)
  4. Scaling horizontal Edge, pas de serveur monolithique

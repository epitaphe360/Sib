Chapitre 1
Premiers pas — Inscription & Connexion

Créer un compte

La plateforme SIB 2026 propose trois types de comptes : Visiteur, Exposant et Partenaire. Le processus d'inscription varie légèrement selon le profil choisi.

ℹ️
Accès à l'inscription
Rendez-vous sur sib2026.ma puis cliquez sur le bouton "S'inscrire" dans la barre de navigation, ou accédez directement à la page de sélection de profil.

Étapes d'inscription — Visiteur

1
Choisir le type de visite
Sélectionnez "Visiteur gratuit" pour un accès de base, ou "Visiteur VIP" (300 EUR) pour l'accès complet avec lettre de visa et badge personnalisé.
2
Remplir le formulaire d'inscription
Prénom, nom, email professionnel, mot de passe, pays, secteur d'activité, intitulé de poste. Tous les champs marqués * sont obligatoires.
3
Vérifier votre email
Un email de confirmation est envoyé à l'adresse indiquée. Cliquez sur le lien de vérification dans les 24h. Vérifiez vos spams si nécessaire.
4
Attendre la validation admin
L'équipe SIB 2026 examine votre dossier. Vous recevrez un email de confirmation d'activation du compte sous 24-48h ouvrées.
5
Accéder à votre espace personnel
Une fois validé, connectez-vous pour accéder à votre tableau de bord personnalisé, votre badge numérique et toutes les fonctionnalités.

Se connecter à la plateforme

Connexion email / mot de passe

1
Accéder à la page de connexion
Cliquez sur "Se connecter" dans la navigation principale.
2
Saisir vos identifiants
Email et mot de passe enregistrés lors de l'inscription.
3
Valider la connexion
Cliquez sur "Se connecter". Vous êtes redirigé vers votre tableau de bord.

Connexion via Google

1
Cliquer sur "Continuer avec Google"
Sur la page de connexion, utilisez le bouton Google OAuth.
2
Choisir votre compte Google
Sélectionnez votre compte Google dans la fenêtre popup.
3
Connexion automatique
Si le compte Google est associé, connexion directe. Sinon, un compte est créé automatiquement.
🔐
Mot de passe oublié ?
Cliquez sur "Mot de passe oublié" sur la page de connexion. Un lien de réinitialisation vous est envoyé par email. Le lien expire après 1 heure.
⚠️
Session automatique
Par mesure de sécurité, votre session expire automatiquement après 30 minutes d'inactivité. Vous serez invité à vous reconnecter. Pensez à sauvegarder vos modifications avant de quitter la page.

Espace Exposant — Profil & Mini-Site

Configurer votre profil exposant

En tant qu'exposant, votre profil est votre vitrine numérique au sein du catalogue SIB 2026. Un profil complet et soigné maximise votre visibilité auprès des visiteurs.

1
Accéder à Mon Profil
Tableau de bord → Mon Profil → Modifier le profil.
2
Informations de base
Nom de l'entreprise, logo (recommandé : PNG 400x400px), description de l'activité, secteur, pays, coordonnées, site web.
3
Certifications et labels
Ajoutez vos certifications ISO, labels qualité ou distinctions sectorielles. Elles apparaîtront sur votre profil et mini-site.
4
Définir vos disponibilités
Configurez vos créneaux de disponibilité pour les rendez-vous B2B via le planning de disponibilités.

Créer votre Mini-Site

Le mini-site est une page web complète dédiée à votre entreprise, accessible depuis le catalogue exposants et via une URL unique personnalisée.

🏗️
Éditeur sans code
L'éditeur drag-and-drop ne nécessite aucune compétence technique. Ajoutez, réorganisez et personnalisez les sections en quelques clics. Votre mini-site est publié instantanément.

Sections disponibles dans l'éditeur

🎯 Section Hero
Image de bannière principale, titre accrocheur, sous-titre et bouton d'action. Première impression de votre marque.
ℹ️ À Propos
Histoire de l'entreprise, valeurs, chiffres clés. Texte riche avec formatage.
📦 Produits
Catalogue de vos produits et services avec photos, descriptions et prix.
📰 Actualités
Vos dernières nouvelles, lancements de produits, communiqués de presse.
🖼️ Galerie
Portfolio photos de vos réalisations, produits, équipe et stand salon.
👥 Équipe
Présentez vos collaborateurs clés avec photo, poste et coordonnées.
🏅 Certifications
Badges et logos de vos certifications ISO, labels et distinctions.
📞 Contact
Formulaire de contact, adresse, numéros de téléphone, carte de localisation.

Personnaliser le thème de votre mini-site

1
Paramètres du thème
Dans l'éditeur, cliquez sur "Thème" pour ouvrir le panneau de personnalisation.
2
Couleurs de marque
Choisissez votre couleur primaire, secondaire et accent. Entrez le code hexadécimal ou utilisez le sélecteur visuel.
3
Publier
Cliquez sur "Publier le mini-site" pour le rendre visible dans le catalogue et sur votre URL.
🔗
URL de votre mini-site
Votre mini-site est accessible à l'adresse :
sib2026.ma/mini-sites/[votre-identifiant]

Partagez ce lien sur vos supports de communication avant le salon.
📊

Chapitre 4
Networking & Rendez-vous B2B

Système de Matching par Intelligence Artificielle

L'algorithme de matching analyse votre profil (secteur, intérêts, objectifs de visite) et le compare à celui des exposants et visiteurs pour vous proposer les rencontres les plus pertinentes.

Comment le matching fonctionne
L'algorithme calcule un score de compatibilité basé sur : le secteur d'activité, les intérêts déclarés, les objectifs de visite, l'historique d'interactions. Plus votre profil est complet, plus les recommandations sont précises. Les intérêts mutuels sont automatiquement détectés pour maximiser les opportunités B2B.

Accéder aux recommandations de matching

1
Tableau de bord → Networking
La section Networking affiche vos recommandations personnalisées triées par score de compatibilité.
2
Explorer les profils suggérés
Cliquez sur un profil pour voir les détails, le score de compatibilité et les intérêts communs.
3
Envoyer un message ou prendre rendez-vous
Depuis le profil, envoyez un message via la messagerie interne ou proposez directement un créneau de rendez-vous.

Prendre un rendez-vous B2B

Réserver un créneau

1
Trouver l'exposant ou visiteur
Via le catalogue exposants, le matching, ou la recherche.
2
Cliquer sur "Prendre rendez-vous"
Disponible sur chaque profil exposant dans le catalogue.
3
Sélectionner le créneau
Le calendrier affiche les créneaux disponibles de l'exposant. Choisissez le jour et l'heure.
4
Confirmer
Saisissez l'objet de la réunion et confirmez. Vous recevrez une confirmation par email.

Gérer vos rendez-vous

📅
Tableau de bord → Mes Rendez-vous
Retrouvez tous vos rendez-vous (à venir, passés, annulés) avec les coordonnées de votre interlocuteur et l'objet de la réunion.
Actions sur un rendez-vous
  • Accepter ou refuser une demande reçue
  • Annuler un rendez-vous confirmé (avec motif)
  • Contacter l'interlocuteur par messagerie
  • Ajouter au calendrier (.ics export)
  • Laisser une évaluation après la réunion

Messagerie Interne

La messagerie interne permet d'échanger directement avec les exposants, visiteurs et membres de l'équipe SIB — sans partager vos informations de contact personnelles.

Démarrer une conversation
Depuis n'importe quel profil, cliquez sur "Envoyer un message". La conversation est visible dans votre espace Messagerie (icône dans la navigation).
Pièces jointes
Partagez des documents, présentations ou fiches produits directement dans la conversation. Formats supportés : PDF, images, Office.

Paiements & Abonnements

Modes de paiement disponibles

💳 PayPal — International
  • Paiement en EUR uniquement
  • Carte bancaire VISA / Mastercard / Amex
  • Compte PayPal existant accepté
  • Confirmation immédiate et automatique
  • Reçu envoyé par email PayPal
  • Sécurisé par chiffrement PayPal
🏦 CMI — Maroc
  • Paiement en MAD uniquement
  • Cartes bancaires marocaines (CIH, Attijariwafa, etc.)
  • Visa et Mastercard émises au Maroc
  • Confirmation immédiate et automatique
  • Reçu envoyé par email de la plateforme
  • Gateway CMI certifié PCI-DSS
📄 Virement bancaire — Manuel
  • Accepté en EUR et MAD
  • Coordonnées bancaires fournies après inscription
  • Upload du justificatif de virement obligatoire
  • Validation manuelle par l'équipe admin
  • Délai de validation : 24-72h ouvrées
  • Notification email dès validation
  • Idéal pour les paiements entreprise
  • Bon de commande disponible sur demande

Procédure de paiement par virement bancaire

1
Choisir "Virement bancaire" dans le flux de paiement
Lors de l'étape de paiement de votre inscription, sélectionnez "Virement bancaire".
2
Recevoir les coordonnées bancaires
Les coordonnées du compte SIB 2026 (IBAN, BIC, banque, référence de virement) sont affichées à l'écran et envoyées par email.
3
Effectuer le virement
Depuis votre banque, effectuez le virement en mentionnant impérativement la référence fournie dans le libellé.
4
Uploader le justificatif
Depuis votre espace → Paiement → Uploader le justificatif. Formats acceptés : PDF, JPG, PNG (max 5 Mo).
5
Attendre la validation
L'équipe SIB 2026 vérifie le virement et active votre compte. Vous recevez un email de confirmation avec votre badge numérique.
⚠️
Important : référence de virement
Toujours indiquer la référence de paiement fournie par la plateforme dans le libellé du virement. Sans cette référence, la validation peut être retardée significativement. La référence ressemble à : SIB2026-[ID]-[DATE]

FAQ & Résolution de Problèmes

Questions fréquentes

Je n'ai pas reçu l'email de confirmation
1) Vérifiez votre dossier Spam / Courrier indésirable. 2) Attendez 5 minutes (délai normal). 3) Si toujours absent, cliquez sur "Renvoyer l'email de confirmation" sur la page de connexion. 4) Contactez le support si le problème persiste.
Mon compte est bloqué après plusieurs tentatives de connexion
Le système bloque temporairement un compte après 5 tentatives échouées (protection anti-brute force). Attendez 15 minutes puis réessayez. Utilisez "Mot de passe oublié" si vous avez oublié votre mot de passe.
Ma session s'est terminée automatiquement et j'ai perdu mes modifications
La session expire après 30 minutes d'inactivité. Pensez à sauvegarder régulièrement en cours d'édition (bouton "Enregistrer" ou Ctrl+S). La plateforme affiche un avertissement 5 minutes avant l'expiration.
Mon paiement par virement est en attente depuis plus de 72h
1) Vérifiez que vous avez bien uploadé le justificatif sur la plateforme. 2) Vérifiez que la référence de virement était mentionnée dans le libellé. 3) Contactez le support à contact@sib2026.ma avec votre justificatif en pièce jointe.
Je ne vois pas les recommandations de matching
Le matching nécessite un profil complet. Complétez au minimum : secteur d'activité, centres d'intérêt, objectifs de visite, et description professionnelle. Plus votre profil est détaillé, plus les recommandations sont précises.
Mon mini-site n'est pas visible dans le catalogue
Vérifiez dans l'éditeur que votre mini-site est bien en statut "Publié". Si c'est le cas et qu'il n'apparaît pas, un délai de 5 à 10 minutes peut être nécessaire pour l'indexation. Contactez le support si le problème persiste après 30 minutes.

Contacter le Support

contact@sib2026.ma

Incluez votre email de compte, la description du problème et des captures d'écran si possible.

Icône de chat en bas à droite de toutes les pages. Disponible pendant les heures ouvrées. Pour les partenaires VIP : support prioritaire avec temps de réponse réduit.

Le sélecteur de langue est disponible dans la navigation principale (en haut de page). Trois langues disponibles : Français (FR) · Anglais (EN) · Arabe (AR avec support RTL). La langue est sauvegardée automatiquement dans votre profil.
